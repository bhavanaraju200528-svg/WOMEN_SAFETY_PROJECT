# 🚨 WOMEN SAFETY PROJECT

## 📌 Overview
An AI-powered women safety system that monitors CCTV camera feeds and detects suspicious or harmful activities. The system alerts the security department in real-time to ensure quick response and prevention.

---

## 🎯 Objective
To enhance women's safety by:
- Monitoring real-time video data
- Detecting dangerous situations using AI
- Sending alerts to authorities instantly

---

## 🧠 Features
- 🎥 Real-time CCTV monitoring
- 🧍‍♀️ Human activity detection
- ⚠️ Suspicious behavior recognition
- 🔔 Alert system for emergencies
- 📊 AI-based decision making

---

## 🛠️ Technologies Used
- Python
- OpenCV
- YOLO (Object Detection)
- Deep Learning
- Machine Learning
- NumPy

---

## ⚙️ Working
1. CCTV camera captures live video
2. Frames are processed using OpenCV
3. YOLO model detects humans and activities
4. AI analyzes behavior patterns
5. If danger is detected → alert is sent
